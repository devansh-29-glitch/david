Getting started

1
Download and install
Download David.dmg. Drag to Applications. Right-click → Open the first time to bypass Gatekeeper.
2
Grant screen recording
System Settings → Privacy & Security → Screen Recording → enable David. Required so David can see your screen.
3
Create your account
Name, email, password. That's it. No API keys. No configuration. David connects to its own AI backend.
4
David is now alive
The orb appears on your screen. Say "David" or press Cmd+Shift+D to interact. David will start helping unprompted when it sees you need it.
